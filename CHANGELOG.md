# Changelog

All notable changes to this project are documented in this file.

## 1.0.0 - 2026-04-07

- Marked the project as the first stable `1.0.0` release.
- Added explicit application version metadata and startup version logging.
- Added automated tests for configuration loading and public IP validation behavior.
- Improved the README so setup, image usage, and troubleshooting match the current app.
- Added version-aware container publishing so release tags can publish versioned GHCR images.
